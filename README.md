Overview:
This project showcases an interactive Tableau dashboard built from an OTT dataset containing movies and shows. The dashboard provides insights into streaming content trends, including genre distribution, ratings over time, platform comparisons, popularity, and engagement. It is designed to help users visually explore OTT data and identify meaningful patterns in the entertainment industry.
Usage:
Use filters to drill down into specific genres or platforms.

Explore KPIs for quick insights.

Analyze rating trends and engagement patterns over time.

Identify top-performing movies and genres.
